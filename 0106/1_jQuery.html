1. 01_제이쿼리소개와문법

제이쿼리란? 자바스크립트의 생산성을 향상시켜주는 라이브러리 언어

라이브러리란? 자주 사용하는 코드들을 재사용할 수 있도록 가공하여 코딩의 효율성을 높여주도록 만든 코드들의 집합을 의미한다.

2. jQuery의 특징

  1. 엘리먼트(html요소)를 선택하는 강력한 기능을 제공
  2. 선택된 엘리먼트를 제어할 수 있는 다양한 함수를 제공
  3. 자바스크립트의 자주 사용하는 기능을 라이브러리로 작성해둔 언어이다.

3. 제이쿼리 라이브러리 연결방법

  1. jQuery라이브러리 파일을 직접 다운받아 html 문서에 연결
  2. CDN(Content Delivery Network) - 서비스를 제공하는 업체로부터 링크 주소를 문서에 연결해 사용

4. 제이쿼리 작성법

  1. ${'대상'}.메소드();
  2. jquery('대상').메소드();

5. 제이쿼리에서 '$'' 기호의 의미?

jQuery에서 $의 역할은 괄호 안에 넣을 요소를 선택하는 역할을 한다.

6. 제이쿼리에서 $기호 다음에 들어가는 대상은?

  1. html 태그 : $('body')
  2. css 선택자 : $('#id명')
  3. javascript 객체 : $(document)

7. 제이쿼리 래퍼(wrapper)

  1. $(document).ready(function(){실행할 내용;})
  2. $(function(){실행할 내용;})
  3. jQuery(function(){실행할 내용;})
  4. $(function(){실행할 내용;})(jQuery)
