01 variable(변수) 복습과 유의사항

  • var : 자바스크립트 es6 문법 이전에 사용하던 변수 키워드
  • let : es6 문법에서 새롭게 나온 키워드 (변경이 가능한 변수)
  • const : es6 문법에서 새롭게 나온 키워드 (변경이 불가능한 상수)

var 키워드의 문제점 : 변수(식별자, 변수명)를 재선언 해도 오류가 없다.

유의사항

  1. 전역변수는 최소한으로 사용한다.
  2. 변수는 함수의 시작부분에서 선언한다.
  3. for(반복)문에서는 카운터 변수를 선언할 때 var 예약어를 사용하지 않는다.
    var i;                          <-- 카운터 변수 선언시 var 사용(X)
    for(i = 1 ; i <= 10 ; i++){
      실행 내용;
    }
