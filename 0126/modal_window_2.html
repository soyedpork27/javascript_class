모달윈도 = 팝업 레이어

모달윈도는 사용자 인터페이스 디자인 개념에서 자식 윈도에서 부모 윈도로 돌아가기 전에 사용자의 상호동작을 요구하는 창을 말한다

모달 윈도우는 부모창을 배경으로 어둡게 하고 자식요소가 화면의 가운데 중앙에 배치하도록 하며, 오른쪽 상단에 '닫기' 버튼을 만들어 닫을 수 있도록 한다.

'오늘 하루 열지 않음' 을 클릭하면 체크박스에 체크되고, '닫기' 버튼 클릭 시 쿠키가 생성되며 모달이 숨겨짐

'오늘 하루 열지 않음'을 클릭하면 모달이 열리지 않도록 하기

제이쿼리 쿠키를 활용한 '더 이상 창 띄우지 않기'

제이쿼리 쿠키 플러그인 주소 : https://plugins.jquery.com/cookie/

쿠키(cookie) : 서버에서 클라이언트에게 특정 정보를 저장하는 데이터

사용자 이름, 아이디, 방문 횟수 등을 기록하여 차후 접속 시 활용

제이쿼리 쿠키 작성법

$.cookie('쿠키이름','쿠키값',{expires : 만료일, path : '저장경로'});
