제이쿼리 문서 객체 모델

객체 편집 메서드는 html 문서 안에서 사용자가 선택한 요소의 앞쪽이나 뒤쪽에 내용을 추가하거나 복제, 삭제 하는 메서드

1. 내용 추가하기(add)

  • append() : 선택한 요소(부모)의 안쪽으로 맨 뒤에 내용을 추가하기
  • prepend() : 선택한 요소(부모)의 안쪽으로 맨 앞에 내용을 추가하기
  • after() : 선택한 요소(형제)의 뒤에 내용을 추가하기
  • before() : 선택한 요소(형제)의 앞에 내용을 추가하기
  • insertBefore() :
  • insertAfter() :

내용추가 메서드는 이메일추가, 쇼핑몰 콘텐츠 '더 보기', 모달윈도, 갤러리 작품 더 보기, 공지사항 더 보기, FAQ 더 보기 등 내용을 추가하고자 할 때 응용할 수 있다.

A insertAfter(B), A insertBefore(B)를 사용하면 A라는 요소의 앞쪽에 B라는 요소를 배치할 수 있고, B 라는 요소의 뒤쪽에 A 요소를 배치할 수도 있다. after(), before() 메서드와 기능은 같지만 사용법이 다르다.

 예)
    $('A').after('B');  : A 요소의 바깥쪽으로 뒤에 B내용을 추가한다.
    $('B').insertAfter('A');  : A요소의 바깥쪽으로 뒤에 B내용을 추가한다.

내용 삭제하기(Remove)

사용자가 선택한 요소를 찾아서 내용만 지우거나, 태그요소 자체를 삭제할 수 있다.

  • empty() : 사용자가 선택한 요소의 내용만 삭제
  • remove() : 사용자가 선택한 요소를 삭제
