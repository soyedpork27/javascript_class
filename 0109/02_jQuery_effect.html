02_jQuery_effect(효과)

효과(effect)란? 사용자가 지정한 html 요소를 선택하여 숨기거나 나타내기, 페이드 효과, 슬라이드 다운, 업, 애니메이션 기능을 사용하기 위한 메소드들이 있다.

  • show() - 선택한 요소를 보이게
  • hide() - 선택한 요소 숨김
  • toggle() - 선택한 요소를 숨기거나/나타냄
  • slideDown() - 선택한 요소를 아래로 펼침
  • slideUp() - 선택한 요소를 위로 접음
  • slideToggle() - 선택한 요소를 아래로 펼치거나/위로 접음
  • fadeIn() - 선택한 요소를 서서히 나타냄
  • fadeOut() - 선택한 요소를 서서히 사라지게 함
  • fadeToggle() - 선택한 요소를 서서히 나타내거나/사라지게
  • fadeTo() - 선택한 요소를 지정한 시간만큼 서서히 사라지게









효과메서드 사용 시 괄호() 안에 시간을 설정하는 값인 'slow','normal(기본값)','fast, 속도와 시간(1/1000=1초) 설정이 가능하다.'

실습예제. '버튼'을 클릭 시 모달 레이어가 뜨고, 닫기 버튼 시 서서히 사라지게 하기
