02자바스크립트 변수선언과 작성법 p.463

변수(variable)란? 변하는 데이터를 저장하기 위한 비워진 메모리 공간이다.

변수에 대입된 값은 항상 마지막에 저장된 데이터이다.

1. 변수의 종류

1-1. 전역변수 : 문서전체에 영향을 주는 변수를 말함

1-2. 지역변수 : 함수(function)내에서 선언되어 사용되는 변수로서 그 함수 지역에서만 사용이 가능한 변수

2. 변수의 선언

변수선언시 'var', 'let', 'const'키워드를 변수명 앞에 붙여주어 선언한다.

자바스크립트 표준 최신버전 es6에서 'let', 'const' 사용을 추천한다.

작성법 ) let 변수명; or let 변수명 = 값;

3. 변수이름 작성시 주의사항

  • 변수명은 반드시 영문 대, 소문자를 구분하여 작성해야 한다.
  • 변수의 첫 글자는 숫자, 기호가 올 수 없다.
  • 변수명 사이에는 스페이스(공백)를 사용할 수 없다.
  • 예약어는 피해서 변수명을 사용해야한다.
  • 변수명에는 특수기호를 사용할 수 없다. (단, '$'와 '_'는 사용가능)
  • 변수명은 의미를 담을 수 있는 이름으로 지정하고, camelcase 표기법으로 작성할 것을 권장한다. 예) imageName

4. 여러개의 단어를 조합하여 이름을 짓는 방법 4가지

  1. dash-case : 주로 html, css 에서 '-'기호로 단어와 단어 사이에 작성
  2. snake_case : 주로 html, css 에서 '_'기호로 단어와 단어 사이에 작성
  3. camelCase : 주로 javascript 에서 사용하며 중간에 대문자로 작성하는 것이 특징
  4. pascalCase : 카멜표기법과 유사하지만 첫번째 단어도 대문자로 표시하는것이 차이점. js 생성자 함수사용시 사용함.
