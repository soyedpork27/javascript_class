제이쿼리 문서객체(DOM)관련 메서드

문서 객체 조작이란? DOM(문서 객체 모델)에서 HTML 문서 안에서 모든 요소들과 내용들을 제이쿼리를 사용하여 글자를 변경하거나 추가, 삭제, 속성변경을 할 수 있는것을 말한다.

문서 내용을 변경하기 위한 메서드

  • .text() : html 요소를 선택하여 내용을 입력하거나 가져오기 위한 메서드 (.innerText="";)
  • .html() : html 요소를 선택하여 html 태그 요소를 추가하거나 가져오기 위한 메서드 (.innerHTML="";)
  • .attr() : html 요소를 선택하여 속성값을 추가하거나 변경하기 위한 메서드(.setAttribute();)
  • .val() : html 요소를 선택하여 input 요소의 속성값을 추가하거나 변경하기 위한 메서드
  • .empty() : 내용지우기
  • .remove() : 요소지우기
  • .addClass() : 클래스 추가하기
  • .removeClass() : 클래스 제거하기
  • .hasClass() : 클래스 존재 여부 확인
  • .toggleClass() : 클래스 토글 적용
  • .appendTo() : 부모내용 마지막 위치로 이동하기
  • .prependTo() : 부모내용 처음 위치로 이동하기

실습 1. 버튼 클릭 시 문서내용이 바뀝니다.

아래 text(); 버튼을 클릭하면 이곳에 텍스트 내용이 변경됩니다.

아래 html(); 버튼을 클릭하면 이곳에 html 태그 요소가 추가되어 변경됩니다.


실습 2. 버튼 클릭 시 속성값이 변경됩니다.

$('대상').attr('속성'); - 속성값을 불러옴

$('대상').attr('속성','값'); - 속성값을 변경

$('대상').attr({'속성1':'값1' ,'속성2':'값2'}); - 속성값 여러개를 선택 후 변경
