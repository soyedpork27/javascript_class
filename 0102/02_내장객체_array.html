02_내장객체(object)_배열(array)

배열(array)객체란?
- 연관된 데이터를 하나처럼 관리하고 싶을 때 사용
- 복소의 데이터를 제어할 수 있는 다양한 함수를 제공한다
- 배열의 함수를 이용한 데이터의 편리한 조작이 가능하다

연관된 데이터 정보 - 과일정보, 회원정보, 쇼핑정보, 예약정보, ..

1. 배열 객체 선언하기

방법 1. let array = new Array(data1, data2, data3, data4, .., dataN);

방법 2. let array = [data1, data2, data3, data4, .. dataN];

예) let food = new Array('사과','감','배','딸기'

예) let food = ['사과','감','배','딸기']

2. 배열 객체 호출방법 - array[index번호]

배열 객체를 호출 할 때에는 인덱스 번호를 사용하는데 인덱스 번호는 0 부터 시작한다.

3. 배열 객체 메서드

concat() - 기존 배열에 다른 배열 추가

filter() - 배열 요소 중에서 주어진 필터링 함수에 대해 true인 요소만 골라 새로운 배열을 만듬

forEach() - 배열의 모든 요소에 대해 주어진 함수를 실행

indexOf() - 주어진 값과 일치하는 값이 있는 배열 요소의 첫 인덱스를 찾아줌.
