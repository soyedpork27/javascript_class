01자바스크립트 연산자

자바스크립트 연산자는 사용자가 조건식을 만들어 원하는 결과를 출력하고자 할 때 사용하는 기호이다.

  • 산술연산자 : +, -, *, /, %(나머지)
  • 문자열 결합연산자 : +(문자 + 문자), (문자 + 숫자)
  • 비교연산자 : >, <, >=, <=, ==, ===(데이터 형식까지 같은지 판별)
  • 대입연산자 : =, +=, -=
  • 증감연산자 : ++(1씩 증감), --(1씩 감소)
  • 논리연산자 : &&, ||, !=(같지 않다)
  • 상향조건연산자 : 조건식? 참일 때 값 : 거짓일 때 값;
  • 타입연산자 : typeof (데이터의 형식을 체크해 준다)
