01_window(객체)_window(브라우저)

브라우저 객체는 사용자가 사용하는 웹브라우저의 창에서 다양한 동작이나 속성을 설정하고자 할 때 사용하는 객체이다.

window 객체

window 객체는 브라우저 객체 중 최상위 객체로서 현재 실행되고 있는 브라우저 창을 나타낸다.

window객체의 속성(프로퍼티)
document
frameElement (잘 사용되지 않음)
innerHeight
innerWidth
location : 현재 브라우저의창의 주소값
name : 브라우저 창의 title
screenX : 브라우저의 좌표값을 알 수 있다
screenY : 브라우저의 좌표값을 알 수 있다
scrollX : 브라우저 스크롤의 좌표값을 알 수 있다
scrollY : 브라우저 스크롤의 좌표값을 알 수 있다
sessionStorage
window객체의 메서드(함수)
alert() - 윈도우 창에서 경고 창(메시지)을 띄울 때 사용한다.
blur() - 현재 창에서 포커스를 제거
close() - 현재 창을 닫음
confirm() - 확인, 취소 버튼이 있는 확인창을 표시
focus() - 현재 창에 포커스를 부여
moveBy() - 현재 창을 지정한 크기만큼 이동
moveTo() - 현재 창을 지정한 좌표로 이동
open() - 새로운 창을 연다
prompt() - 사용자가 입력 가능한 창을 연다
resizeTo() - 브라우저의 창 크기를 동적으로 조정
scroll() - 문서의 특정 위치로 스크롤 중요 사용할듯
sizeToContent() - 내용에 맞게 창의 크기를 조정
stop() - 로딩 중지
모달 윈도(modal window)
모달윈도는 사용자 인터페이스(UI) 디자인 개념에서 자식 윈도우에서 부모 윈도우로 돌아가기 전에 사용자의 상호동작을 요구하는 창을 말한다
모달 윈도우는 부모창을 배경으로 어둡게 하고 자식요소가 화면의 가운데에 배치하도록 하며, 오른쪽 상단에 '닫기' 버튼을 만들어 닫을 수 있도록 한다.
